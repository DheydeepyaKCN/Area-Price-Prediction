# 🏠 Area Price Prediction using Linear Regression

A Machine Learning project that predicts the price of a house based on its area using the Linear Regression algorithm. The model is trained using Scikit-learn and deployed as an interactive web application using Streamlit.

---

## 📌 Project Overview

This project demonstrates the complete Machine Learning workflow, including data preprocessing, visualization, model training, prediction, model serialization, and deployment.

Users can enter the area of a house through a simple Streamlit interface, and the model predicts the estimated price instantly.

---

## 🚀 Features

- Predicts house price based on area
- Interactive Streamlit web application
- Linear Regression model using Scikit-learn
- Model saved using Pickle
- Simple and user-friendly interface
- Fast real-time prediction

## ⚙️ Machine Learning Workflow

1. Load the dataset
2. Split data into independent and dependent variables
3. Visualize the data
4. Train a Linear Regression model
5. Make predictions
6. Save the trained model using Pickle
7. Deploy the model using Streamlit

## ▶️ Run Locally

Clone the repository

```bash

```

Install dependencies

```bash

```

Run the application

```bash
streamlit run app.py
```
